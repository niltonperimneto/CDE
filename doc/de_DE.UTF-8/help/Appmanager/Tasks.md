# Starten einer Anwendung
Starten einer AnwendungAnwendungen:StartenAnwendungsmanager:Öffnen

Um den Anwendungsmanager zu öffnen, auf das Bedienelement für den Anwendungsmanager
im Bedienfeld klicken.

Um die Anwendungsgruppe mit der Anwendung zu öffnen, doppelt auf das
entsprechende Symbol klicken.

AnwendungsgruppeSymbole für Anwendungsgruppen sind spezielle Ordner, deren Verhalten
dem von Ordnersymbolen im Dateimanager entspricht.

Die Anwendungsgruppe enthält Symbole, die die Anwendung sowie der
Anwendung zugeordnete Objekte darstellen, wie beispielsweise Informationsdateien
(README) und Schablonen.

AnwendungssymboleAktionssymboleDas Symbol, mit dem die Anwendung gestartet wird, wird alsAnwendungssymboloderAktionssymbolbezeichnet.

Doppelt auf das Aktionssymbol für die Anwendung klicken.

In manchen Fällen kann die Anwendung auch gestartet werden, indem eine
Datendatei zum Aktionssymbol gezogen und übergeben wird.

# Aktualisieren des Anwendungsmanagers
Anwendungsmanager:aktualisierenErneutes Laden von AnwendungenAnwendungen:erneut laden

Der Anwendungsmanager wird beim ersten Anmeldevorgang erstellt und bei
jedem weiteren Anmeldevorgang erneut erstellt. Bei jedem erneuten Erstellungsvorgang
werden bestimmte Positionen innerhalb des Systems und des Netzes nach
Anwendungen durchsucht; diese werden dann in den Anwendungsmanager gestellt.

Fügt der Systemadministrator eine Anwendung hinzu, kann der Anwendungsmanager
aktualisiert werden. Hierzu wie folgt vorgehen:

Ab- und erneut anmelden.

Oder:Die Anwendungsgruppe 'Desktop-Werkzeuge' öffnen und
doppelt auf 'Anwendungen erneut laden' klicken.
# Erstellen einer persönlichen Anwendungsgruppe
Anwendungsgruppe:persönlichAnwendungsmanager:ändern

Eine persönliche Anwendungsgruppe ist eine Gruppe, deren einziger Eigentümer der
jeweilige Benutzer ist. Der Benutzer erstellt die Gruppe selbst und
verfügt über die Berechtigung, sie zu ändern.

Ein Verzeichnis erstellen:`Standardverzeichnis`/.dt/appmanager/`Anw.gruppenname`

Dabei istAnw.gruppennameder Name der Anwendungsgruppe, der auch
später im Anwendungsmanager verwendet wird.

Die Anwendungsgruppe 'Desktop-Werkzeuge' öffnen und doppelt auf
'Anwendungen erneut laden' klicken.

Die neue Anwendungsgruppe wird nun auf der obersten Ebene des
Anwendungsmanagers angezeigt.

Die persönliche Anwendungsgruppe kann auf die gleiche Weise verwendet
werden wie die übrigen Verzeichnisse, auf die der Benutzer Schreibzugriff
hat. Dateien können in die persönliche Anwendungsgruppe verschoben oder
kopiert werden, Unterverzeichnisse können erstellt und nicht mehr benötigte
Dateien gelöscht werden.

# Hinzufügen einer Anwendung zu einer persönlichen Anwendungsgruppe
Anwendungsmanager:ändernAnwendungen:hinzufügen

Mit dieser Prozedur ist folgendes möglich:

Erstellen einer Aktion für die Anwendung

Eine Aktion ist eine Desktop-Konstruktion, deren Verhalten dem eines
Makros entspricht - sie ermöglicht die automatische Ausführung von
Aufgaben, die normalerweise über Befehlszeilen ausgeführt werden.
Das Desktop verwendet Aktionen, um Befehlszeilen mit Symbolen zu verbinden.

Erstellen eines Aktionssymbols, das die Anwendung in der Anwendungsgruppe
darstellt

Falls erforderlich, eine Anwendungsgruppe erstellen, auf die Schreibzugriff
besteht. Siehe hierzu.

Die Befehlszeile feststellen, die normalerweise zum Starten der Anwendung
in einem Terminal-Emulatorfenster verwendet wird.

Aktion erstellen

Mit der Desktop-Anwendung 'Aktion erstellen' eine Aktion erstellen.
Sieh hierzu auchVerwendung
von 'Aktion erstellen'.

Für die Verwendung von 'Aktion erstellen' muß ein Aktionsname eingegeben
werden. Der vom Benutzer eingegebene Name wird als Aktionsbezeichnung
verwendet.

Mit 'Aktion erstellen' wird ein Aktionssymbol im Standardverzeichnis erstellt.
Das Symbol besteht aus einer ausführbaren Datei, deren Name dem
Namen der Aktion entspri cht.

Das erstellte Aktionssymbol vom Standardverzeichnis des Benutzers
in die persönliche Anwendungsgruppe verschieben oder kopieren.
# Registrieren von Anwendungen im Anwendungsmanager
Anwendungen:registrierenRegistrieren von AnwendungenAnwendungsmanager:ändern

Wird eine Anwendung im Anwendungsmanager registriert, verfügt sie über
eine eigene Anwendungsgruppe. Diese Anwendungsgruppe steht allen Benutzern
auf dem System zur Verfügung.

Das Registrieren von Anwendungen ist eine Systemverwaltungsaufgabe.
Anweisungen hierzu sind im HandbuchCommon Desktop Environment - Systemverwaltungenthalten.
