
# Aktion erstellen - Hilfe
Aktion erstellen - Hilfe

Bei 'Aktion erstellen' handelt es sich um eine Anwendung, mit der
folgende Komponenten erstellt werden können:

EineAktionzum Starten einer Anwendung

EinDatentypfür die Datendateien einer Anwendung

Aktionen zum Öffnen und Drucken der Datendateien der Anwendung

'Aktion erstellen' ist außerdem sinnvoll bei der Erstellung einfacher Aktionen
für Shell-Scripts.
# Ausführen von 'Aktion erstellen'
Aktion erstellen:startenAktion:erstellenDatentyp:erstellen

Die Anwendungsgruppe 'Desktop-Anwendungen' im Anwendungsmanager öffnen und
doppelt auf 'Aktion erstellen' klicken.
