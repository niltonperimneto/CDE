# Impossibile creare <file o cartella>


Causa possibile: Non si possiede l'autorizzazione di scrittura per la cartella in cui si desidera creare il file.

Soluzione: Cambiare le autorizzazioni della cartella: selezionare la cartella e scegliere Modificare autorizzazioni dal menu Selezioni.
