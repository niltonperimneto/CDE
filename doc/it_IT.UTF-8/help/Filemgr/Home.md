* **Menu** 

Contiene i comandi della Gestione di file
* **Percorso a icone** 

Mostra la posizione della cartella visualizzata
* **Percorso esplicito** 

Indica la posizione della cartella corrente
* **Riga di stato** 

Mostra il percorso completo della cartella visualizzata
* **Area di visualizzazione** 

Mostra il contenuto della cartella visualizzata
* **Icone degli oggetti** 

Ogni icona rappresenta un file o una cartella della cartella corrente
* **Riga dei messaggi** 

Questa riga indica quanti oggetti sono contenuti nella cartella
visualizzata. Visualizza inoltre i messaggi relativi alle operazioni
della Gestione di file.

# Per avviare la Gestione di file


Fare clic sul controllo della Gestione di file nel Pannello principale.
