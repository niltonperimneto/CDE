
# Aiuto sull'Agenda
Aiuto sull'AgendaAgenda, usoavvio dell'Agendaintroduzione

L'Agenda del desktop permette di programmare appuntamenti personali e di gruppo, di creare liste di impegni e di appuntamenti e di impostare promemoria per ricordarli.
# Per avviare l'Agenda


Fare clic sul controllo dell'Agenda nel Pannello principale.

&newline;&empty;

Quando viene avviata per la prima volta, l'Agenda visualizza il mese corrente.Per una panoramica della finestra dell'Agenda, fare clicin questo punto.
