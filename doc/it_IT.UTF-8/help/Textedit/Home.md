# Per aprire l'Editor di testo:




Fare clic sul controllo dell'Editor di testo nel Pannello principale.

&sigspace;

Oppure, avviare l'Editor di testo usando uno dei metodi seguenti:

Aprire il pannello secondario Applicazioni personali dal Pannello principale e fare clic sul controllo dell'Editor di testo.

&sigspace;

&sigspace;

Trascinare l'icona di un file di testo e rilasciarla sul controllo dell'Editor di testo nel Pannello principale.

Fare doppio clic su un file di dati nella Gestione di file.

Selezionare un file di dati nella Gestione di file e scegliere Aprire dal menu Selezioni.

In una finestra di terminale, digitare il comando seguente:dtpad`nomefile`&
