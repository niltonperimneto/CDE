
# Aiuto su Creare azione
Aiuto su Creare azione

Creare azione è un'applicazione che permette di creare:

Un'azioneper l'avvio di un'applicazione

Untipo di datiper i file di dati di un'applicazione

Una serie di azioni per aprire e stampare i file di dati di un'applicazione

L'applicazione può anche essere usata per creare azioni semplici per gli script delle shell.
# Per avviare Creare azione
Creare azione:avvioazione:creazionetipo di dati:creazione

Aprire il gruppo di applicazioni Applicazioni_desktop nella Gestione di applicazioni e fare doppio clic su Creare azione.
