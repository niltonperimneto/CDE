
# Aiuto sulla Gestione posta
Aiuto sulla Gestione postaGestione posta, usouso della Gestione postaintroduzione

La Gestione posta permette di inviare, ricevere e gestire i messaggi di posta elettronica. Quando si riceve un messaggio elettronico (email), è possibile:

Salvare il messaggio in una casella postale all'interno del proprio file system

Rispondere al mittente del messaggio

Inoltrare il messaggio ad un altro destinatario insieme a un proprio messaggio

Cancellare il messaggio

Stampare il messaggio

È anche possibile inviare e ricevere allegati in vari formati: immagini, documenti o file eseguibili.

Il metodo più semplice per inserire un allegato in un messaggio consiste nel trascinare e rilasciare il file da un'altra applicazione, ad esempio la Gestione di file.
# Per aprire la Gestione posta


Fare clic sul controllo della Gestione posta nel Pannello principale.
