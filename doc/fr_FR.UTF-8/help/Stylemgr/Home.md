
# Gestionnaire de configuration - Aide


Le Gestionnaire de configuration permet de modifier les caractéristiques
visuelles du Bureau (palette de couleurs, polices, etc.) et le comportement
des éléments système tels que la souris ou le clavier, afin de personnaliser
l'environnement en fonction de vos préférences.






# Pour lancer le Gestionnaire de configuration :


Cliquez sur la commande Gestionnaire de configuration du Tableau de bord.
