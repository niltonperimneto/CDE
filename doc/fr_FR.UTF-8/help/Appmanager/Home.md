
# Gestionnaire d'applications - Aide
Gestionnaire d'applications - Aide

Le Gestionnaire d'applications est un conteneur dans lequel
les applications et outils installés sur le système sont stockés.

Le niveau supérieur du Gestionnaire d'applications contient un
ensemble de groupes d'applications.

Un groupe d'applications est un dossier spécial contenant une
application et, éventuellement, certains fichiers utiles (fichiers
de données types, modèles et fichiers readme, par exemple).

Les groupes du Gestionnaire d'applications sont fournis en
standard ou installés par l'administrateur système. Ils peuvent
se trouver sur le système local ou sur un système éloigné.
