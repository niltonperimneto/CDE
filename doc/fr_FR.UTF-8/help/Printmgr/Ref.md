lorsqu'une icône est sélectionnée.Zone de travailCette zone, également appelée "vue" affiche les imprimantes
et les travaux d'impression.Ligne de messageElle se situe sous la zone de travail et contient les informations suivantes
:le nombre d'imprimantes auxquelles
vous pouvez envoyer des travaux d'impression;le nombre d'imprimantes qui n'apparaissent
pas dans la vue;l'affichage ou non de travaux d'impression
dans la vue;la date de mise à jour des
informations sur les imprimantes (éventuellement).Fenêtre des travaux d'impressionIl existe trois zones sur la fenêtre principale:Barre de menusIl s'agit de la zone supérieure contenant les menus déroulants
de l'application Travaux d'impression. Notez que le menu Sélectionné
est actif uniquement lorsqu'une icône est sélectionnée.Zone de travailCette zone, également appelée "vue", affiche les imprimantes
et les travaux d'impression.Ligne de messageCette ligne d'information se situe sous la zone de travail. Elle contient
les informations suivantes:l'état de l'imprimante;les dates de mise à jour des
informations sur les imprimantes (éventuellement)..Objet ImprimanteCette
icône représente une imprimante reconnue par le système.Le
signal d'erreur s'affiche lorsque l'imprimante détecte un incident,
tel qu'un bourrage papier. Il s'affiche également lorsque le système
ne peut pas communiquer avec une imprimante éloignée. Pour plus
de détails, cliquez sur cette icône.Le
bouton plus (+) permet d'ouvrir l'icône d'une imprimante; cliquez dessus
pour afficher les travaux de cette imprimante (uniquement disponible à
partir du Gestionnaire d'impression).Le
bouton moins (-) permet de fermer l'icône d'une imprimante; cliquez
dessus pour masquer les travaux de cette imprimante (uniquement disponible
à partir du Gestionnaire d'impression).Voir aussi
