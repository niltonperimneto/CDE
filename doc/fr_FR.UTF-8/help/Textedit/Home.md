
# Editeur de texte - Aide
Editeur de texte - Aide

L'Editeur de texte vous permet de créer et d'éditer des documents courts,
tels que des notes, des messages ou des fichiers ressources.

# Ouverture de l'Editeur de texte :




Cliquez sur l'icône Editeur de texte du Tableau de bord.

Vous pouvez également lancer l'Editeur de texte en utilisant l'une des
méthodes suivantes:

Ouvrez le tableau secondaire Applications personnelles du Tableau
de bord et cliquez sur l'icône Editeur de texte.

Faites glisser une icône de fichier sur celle de l'Editeur de texte,
dans le Tableau de bord.

Cliquez deux fois sur une icône de fichier de données, dans le
Gestionnaire de fichiers.

Sélectionnez une icône de fichier de données dans le Gestionnaire de
fichiers, puis l'option Ouvrir du menu Sélectionné(s).

Tapez la commande suivante dans une fenêtre de terminal:dtpad`nom_fichier`&
