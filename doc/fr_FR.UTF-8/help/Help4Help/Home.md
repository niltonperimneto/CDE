
# Aide sur l'aide
Aide sur l'aide

Les informations d'aide sont organisées en sujets, que vous pouvez
choisir et afficher dans une fenêtre d'aide.
# Pour sélectionner un sujet :


sélectionnez un titre dans la liste des sujets affichée dans la partie
supérieure de la fenêtre d'aide;

ou,choisissez un hyperlien dans la zone d'affichage de
la fenêtre d'aide.

Unhyperlienest un mot, une phrase ou un graphique actif
qui permet d'afficher des informations complémentaires. Dans une fenêtre
d'aide, tout texte souligné est un hyperlien. Une boîte grise ouverte
identifie un hyperlien graphique.
