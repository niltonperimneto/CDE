
# Emulateur de terminal dtterm - Aide
Emulateur de terminal dtterm - AideLancement: émulateur de terminal

Un émulateur de terminal est une application qui affiche une fenêtre
dans laquelle vous pouvez entrer des commandes UNIX pour transférer
du texte (par couper-coller) entre des fenêtres d'émulation de terminal
ou à l'intérieur de telles fenêtres.

L'invite de ligne de commande est un caractère spécial
(%,<ou$, par exemple) qui apparaît dans la fenêtredttermlorsque vous appuyez surEntrée.
Une case ou une barre de petites dimensions, appelée "curseur",
indique à quel emplacement de la fenêtre les caractères
entrés au clavier apparaîtront.

L'invite de ligne de commande comme le curseur s'affichent
dans les fenêtres du terminal.
&ProductName; fournit, pas défaut, l'émulateur de terminaldtterm.
Dans la fenêtre qu'il affiche, les invites de ligne et commande et
les touches de fonction présentent le même fonctionnement que sur
un terminal DEC VT220.
L'émulateur de terminaldttermémule la partie du terminal VT220
qui est en conformité avec les normes ANSI et ISO.
