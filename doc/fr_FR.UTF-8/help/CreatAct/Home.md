
# Créer une action - Aide
Créer une action - Aide

L'application Créer une action permet de créer:

uneactionpermettant de lancer une application;

untype de donnéespour les fichiers
de données de l'application;

des actions permettant d'ouvrir et d'imprimer les fichiers de données
de l'application.

Elle permet également de créer des actions simples associées aux
scripts shells.
# Pour lancer Créer une action :
Créer une action:lancementAction:créationType de données:création

Ouvrez le Gestionnaire d'applications et cliquez deux fois sur Créer
une action dans le groupe Applications du Bureau.
