
# Tareas del Editor de iconos

# Introducción al Editor de iconos





































# Manipulación de imágenes del Editor de iconos






















Estas instrucciones suponen que se está utilizando el ratón. La mayoría
de tareas también pueden llevarse a cabo sin ratón.
Para más información, consulte el apartadoMétodos
abreviados del teclado para el escritorio.
# Para iniciar el Editor de iconos
inicio: Editor de iconosEditor de iconos: inicio

Pulse el control de desplazamiento situado encima del control del Editor
de textos del Panel frontal.

En el subpanel de Aplicaciones personales, pulse el control del Editor
de iconos.

El Editor de iconos empieza con un área de dibujo vacía.
# Modos adicionales de iniciar el Editor de iconos
apertura: icono para editariconos: apertura para editarEditor de iconos: abrir icono para editar

El Editor de iconos también se puede iniciar con un icono ya cargado:

En una ventana abierta del Gestor de archivos, pulse dos veces el
archivo de iconos para abrirlo.

O biendeje caer el archivo de iconos en el control del Editor
de iconos en el subpanel de Aplicaciones personales.

O bienseleccione Editor de iconos en las Aplicaciones de escritorio
del Gestor de aplicaciones.
# Para dibujar una imagen
dibujar imagen de iconosiconos: dibujo

Seleccione una herramienta de dibujo (como un lápiz, un círculo o una
línea).

Seleccione un color (un color estático, un gris estático o un color
dinámico).

Efectúe una pulsación sobre el área de dibujo y empiece a dibujar con
la herramienta que haya seleccionado.
# Consulte también





# Para dibujar una imagen rellenada
dibujo de imagen rellenadaiconos: dibujo de imagen rellenada

Seleccione la casilla de verificación de Rellenar figuras situada
debajo de la paleta de herramientas.

Las herramientas del rectángulo, del polígono, del círculo y de
la elipse cambian el contorno de dichas figuras por las figuras
en la paleta de herramientas.

Seleccione una herramienta de dibujo de figuras.

Seleccione un color (un color estático, un gris estático o un color
dinámico).

(Consulte.)

Empiece a dibujar con la herramienta que ha seleccionado.

Cuando libera el botón para acabar de dibujar el objeto, todos los pixels
que se encuentran dentro de los límites del objeto se vuelven del color
seleccionado.

# Para trazar líneas o pixels a pulso
línea: trazado

Seleccione la herramienta Lápiz de la paleta.

Para trazar pixels sueltos, pulse sobre el área de dibujo.

Para trazar líneas a pulso, pulse sobre el área de dibujo y
arrastre el ratón.
Libere el botón para acabar la línea.

Libere el botón para dibujar la línea.

# Para borrar un área
borrar: dibujo
# Para borrar un área extensa


Seleccione la herramienta Borrar de la paleta.

Para borrar, pulse el ratón y arrástrelo en el área de dibujo.

Libere el botón para acabar de borrar.
# Para borrar un área pequeña


Seleccione la herramienta Lápiz de la paleta.

En Colores dinámicos, seleccione Transparente.

Pulse sobre el área de visualización.

Cada pixel que se pulsa se fija en transparente, acción que,
efectivamente borra el pixel.
# Consulte también





# Para llenar un área con un color
color: dibujo

Seleccione la herramienta Rellenar de la paleta.

Seleccione un color estático, gris estático o color dinámico.
(Consulte.)

Pulse un pixel en el área que debe llenarse.

El pixel que se ha pulsado y todos los pixels contiguos del mismo
color, cambian de color y se vuelven del color seleccionado. El
área rellenada se extiende en todas las direcciones y está limitada
sólo por pixels de colores diferentes del que ha pulsado.
# Consulte también





# Para dibujar líneas conectadas


Seleccione la herramienta Polilínea de la paleta.

Pulse en el punto inicial de la primera línea del área de dibujo.

Pulse al final de cada segmento de la línea.

Pulse al final del último segmento de la línea.
# Consulte también





# Para dibujar un polígono
Polígono: dibujo

Seleccione la herramienta Polígono de la paleta.

En el área de dibujo, pulse en el punto inicial del primer segmento de
línea.

Pulse al final de cada segmento de la línea.

Pulse al final del segmento de la última línea para cerrar el
polígono.

Una línea se dibuja desde el final del segmento de la última línea al
punto de inicio del segmento de la primera línea.
# Consulte también







# Para dibujar una elipse
elipsis: dibujo

Seleccione la herramienta Elipse de la paleta.

Pulse el ratón sobre el área de dibujo y arrástrelo para trazar el
contorno de la elipse.

Libere el botón para acabar la elipse.
# Consulte también







# Para deshacer una operación de dibujo


Elija Deshacer en el menú Editar.

Sólo puede deshacerse el cambio más reciente.
# Para abrir un icono ya existente


Seleccione Abrir del menú Archivo.

Si tiene cambios no guardados, el Editor de iconos le avisará de que
cargando otro icono, se perderán los cambios. Pulse Cancelar para
cancelar el mandato Abrir. Pulse OK si no desea guardar los cambios.

Escriba el nombre del archivo de iconos que desea cargar o
selecciónelo de la lista.

Pulse Abrir.

Al cargar un archivo, el formato de salida se establece
automáticamente en el formato del archivo que carga.
# Consulte también



# Para guardar un icono
Editor de iconos: guardar icono

Elija un formato de salida en el menú Opciones:

XBM para el formato de mapa de bits X de dos colores

XPM para el formato de mapa de pixels X multicolor (predeterminado)

Elija Guardar en el menú Archivo.

Si el icono no tiene nombre o selecciona Guardar como, el Editor de
iconos le solicitará un nombre de archivo. Escriba un nombre de archivo
y luego pulse OK.
# Consulte también







# Para Guardar un icono con un nombre nuevo
Editor de iconos: guardar un icono con un nombre nuevo

Seleccione Guardar como en el menú Archivo.

Escriba el nombre del archivo con el que desea guardar el icono.

Pulse Guardar.
# Consulte también



# Para iniciar un nuevo icono


Seleccione Nuevo en el menú Archivo.

Si tiene cambios no guardados, el Editor de iconos le avisará de que
iniciando un nuevo icono, se perderán los cambios. Pulse Cancelar para
cancelar el mandato Nuevo. Pulse OK si no desea guardar los cambios.

El Editor de iconos borra el área de trabajo y redimensiona su tamaño hasta
32x32, que es el predeterminado.
El tamaño máximo para un icono es 256&times;256.
# Consulte también





# Para redimensionar el icono actual
Editor de iconos: redimensionar iconosiconos: redimensionar

En el menú Editar, elija Redimensionar icono.

Edite la anchura y la altura actuales en el cuadro de dialogo para
especificar el nuevo tamaño.

Pulse OK.

Si disminuye el tamaño del icono, los extremos inferior y derecho se
recortan. Si aumenta el tamaño del icono, la imagen actual
permanece en la parte superior izquierda del icono.

# Para seleccionar un área del icono
iconos: selección de partes

Seleccione la herramienta Selección (parte inferior derecha de la
paleta de herramientas).

Pulse el botón 1 del ratón y arrastre un cuadro alrededor del área
que desea seleccionar.
Se resalta un cuadro alrededor del área seleccionada.
# Consulte también













# Para copiar una zona del icono


Seleccione la herramienta Seleccionar (parte inferior derecha de la
paleta).

Pulse el botón 1 del ratón y arrastre un cuadro alrededor del área
que desea copiar.
Se resalta un cuadro alrededor del área seleccionada.

Seleccione Copiar área del menú Editar.

Seleccione Pegar área del menú Editar.

Sitúese en el lugar en el que desea pegar la copia y,
a continuación, efectúe la pulsación.

Para pegar varias copias de la zona, repita los pasos 3 y 4 las veces
que desee.
# Consulte también





# Para cortar una zona del icono


Seleccione la herramienta Seleccionar (parte inferior derecha de la paleta).

Pulse el botón 1 del ratón y arrastre un cuadro alrededor del área
que desea cortar.
Se resalta un cuadro alrededor del área seleccionada.

Seleccione Cortar área en el menú Editar.

Cuando corta una parte de un icono, el área expuesta se vuelve
"transparente" (es decir, se rellena de color Transparente).

Después de cortar una zona, puede volverla a pegar a la imagen
utilizando el mandato Pegar del menú Editar.

# Para redimensionar (escalar) una zona del icono


Seleccione la herramienta Seleccionar (parte inferior derecha de la paleta).

Pulse el botón 1 del ratón y arrastre un cuadro alrededor del área
que desea redimensionar.
Se resalta un cuadro alrededor del área seleccionada.

Pulse el botón 1 del ratón y arrastre un cuadro que defina el

Este cuadro determina cómo se escalará la parte seleccionada
de la imagen;nodetermina dónde se situarán los resultados.

a continuación, efectúe la pulsación.

# Para alternar (crear una imagen especular) una zona del icono
alternar iconoiconos: alternar

Seleccione la herramienta Seleccionar (parte inferior derecha de la paleta).

Pulse el botón 1 del ratón y arrastre un cuadro alrededor del área
que desea alternar.
Se resalta un cuadro alrededor del área seleccionada.

# Para añadir un punto de actuación
punto de actuación en un iconoiconos: punto de actuaciónpunto de actuación: añadirpunto de actuación: suprimir

En el menú Editar, elija Añadir punto de actuación.

Pulse sobre el pixel en el que desea asignar el punto de actuación.

Las imágenes del bitmap (formato XBM) pueden utilizarse como formas
de puntero de ratón especiales.
El punto de actuación marca un sólo pixel que es el "punto" real del puntero.
# Para suprimir un punto de actuación
punto de actuación en un iconoiconos: punto de actuaciónpunto de actuación: supresión

Seleccione el punto de actuación que desee suprimir.

En el menú Editar, elija Suprimir punto de actuación.
