
# Ayuda para el editor de iconos
Ayuda para el editor de iconos

El Editor de iconos es un programa simple de dibujo diseñado
especialmente para crear iconos e imágenes de fondo.
El Editor de iconos sirve para crear iconos de varios colores o mapas
de bits tradicionales en blanco y negro.
# Modo de iniciar el Editor de iconos


Pulse el control de desplazamiento situado encima del control del
Editor de textos del Panel Frontal.

En el subpanel de Aplicaciones personales, pulse el control del
Editor de iconos.
