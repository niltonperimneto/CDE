
# Ayuda del Editor de textos
Ayuda del Editor de textos

El Editor de textos sirve para crear y editar documentos breves, como
memorias, mensajes de correo electrónico o archivos de recursos.Editor de textos: inicioinicio: Editor de textosabrir: Editor de textosControl Panel Frontal, Editor de textosEditor de textos: control Panel frontal
# Para abrir el Editor de textos:




Pulse el control Editor de textos en el Panel Frontal.

También se puede iniciar el Editor de textos con uno de los métodos
siguientes:

Abra el subpanel Aplicaciones personales en el Panel Frontal y
pulse el control Editor de textos.

Arrastre un icono de archivo al control Editor de textos en el
Panel Frontal.

Pulse dos veces un icono de archivo de datos en el Gestor de
archivos.

Seleccione un icono de archivo de datos en el Gestor de archivos y
elija Abrir del menú Seleccionado.

Escriba el mandato siguiente en una ventana de terminal:dtpad`nombrearchivo`&
