# Ventana de Ayuda general
ventana de ayuda generalventana, ayuda general

Una ventana de ayuda general dispone de una barra de menús, un árbol de
temas y un área de visualización de temas.
El árbol de temas lista los temas de ayuda que puede elegir. La parte
inferior de la ventana&emdash;el área de visualización de
temas&emdash;muestra el tema seleccionado.

Todo texto subrayado es un hiperenlace.
Puede utilizar los hiperenlaces para &dquote;saltar&dquote; a
otros temas.
ConsulteSeleccionar un
hiperenlacepara obtener la descripción de los hiperenlaces.

* **Retroceder** 

Vuelve a un tema anterior.

Lista los temas y los volúmenes de ayuda que ha
consultado y le permite volver a visualizar un tema.

búsqueda para encontrar ayuda sobre un tema.

# Cuadro de diálogo Confirmación de enlace de ejecución
Ayuda, cuadro de diálogo Confirmación de enlace de ejecuciónhiperenlaces, política de enlace de ejecuciónenlace de ejecución, política

Un hiperenlace de ejecución ejecuta un mandato en lugar de mostrar
un tema de ayuda. Dado que los enlaces de ejecución interactua con
el sistema, puede elegir cómo desea que el Sistema de ayuda responda
cuando se selecciona un enlace de ejecución.

El comportamiento predeterminado del Sistema de ayuda es visualizar
un cuadro de diálogo de confirmación.

La política de ejecución se controla mediante el recurso de widget
del diálogo AyudaDtNexecutionPolicy. El valor
predeterminado eshelp_execute_query_unaliased.

Para cambiar el comportamiento predeterminado, puede añadir el
recurso de política de ejecución al archivo.Xdefaultspara la aplicación que desea cambiar.

El conjunto de valores de recurso es:

* **help_execute_all** 

Ejecuta todos los enlaces de
ejecución sin mostrar ningún cuadro de diálogo de confirmación.
* **help_execute_query_unaliased** 

Muestra un
cuadro de diálogo sólo para los enlaces sin alias de ejecución
definidos. Un alias de ejecución es un recurso que asigna un nombre
al mandato ejecutado por un enlace de ejecución.
* **help_execute_query_all** 

Muestra un cuadro de
diálogo para todos los enlaces de ejecución
* **help_execute_none** 

Inhabilita todos los
enlaces de ejecución, pero muestra el mandato de ejecución en un
cuadro de diálogo

# Ejemplo


Para inhabilitar todos los enlaces de ejecución de su aplicación,
siga estos pasos:

Cree un archivo.Xdefaultsen el directorio de
inicio del usuario o edite el archivo.Xdefaultsactual para incluir este recurso:`nombre_aplicación`*executionPolicy:help_execute_none

Sustituya elnombre_aplicaciónpor el nombre de la
aplicación que desea cambiar. Si no está seguro de cuál es el nombre
de la aplicación, consulte la documentación de su aplicación.

Inicie la aplicación.

Para probar los cambios, elija el menú de Ayuda de la
aplicación y visualice un tema que contenga un enlace de
ejecución.

Antes de modificar la política de ejecución para un volumen de
ayuda de la aplicación, revise cuidadosamente los mandatos de enlace
de ejecución y la interacción de éstos con su sistema.

&sigspace;
Para obtener más información sobre la política de ejecución del
Sistema de ayuda, consulte la publicaciónHelp System Author's and Programmer's Guide.

Puede cambiar el tamaño de ventana predeterminado de las

de la aplicación. Si ha modificado uno de los tipos de ventana de

# Mensajes de error de la Ayuda
mensajes de error

Si el Sistema de ayuda no puede encontrar un volumen de ayudas, se
muestra este mensaje:

Las posibles causas son:

El volumen de ayudas de la aplicación no está instalado

La ruta de búsqueda de ayuda del escritorio no incluye el
directorio en el que se ha instalado el volumen de ayudas.

Consulte al administrador del sistema para verificar si el volumen de
ayudas de la aplicación se ha instalado y si las rutas de búsqueda de
ayuda están establecidas correctamente.

Si el Visor de ayuda no puede encontrar el ID de un tema de ayuda
en un volumen de ayudas, muestra un mensaje como este:

Las posibles causas son:

Un hiperenlace apunta a un tema que no existe o cuyo ID se ha
cambiado

Un hiperenlace apunta a un ID válido en otro volumen de ayudas
que no está instalado

Consulte al administrador del sistema para verificar si el volumen de
ayudas de la aplicación está instalado de forma adecuada y si las
rutas de búsqueda de ayuda están establecidas correctamente.

Si en el volumen de ayudas en tiempo de ejecución
(archivo.sdl)
existe algún error, el Visor de ayuda muestra este mensaje:

Posibles causas:

Puede que el volumen de ayudas en tiempo de ejecución esté
incompleto o contenga un error.
Consulte al administrador del sistema para comprobar el volumen de
ayudas.
