
# Ayuda del Gestor de estilos


Con el Gestor de estilos, puede personalizar fácilmente los elementos visuales
y el comportamiento del sistema del escritorio. Elija o modifique la paleta de
colores, seleccione un font, cambie el comportamiento del ratón, o lleve a cabo
otras tareas que adapten el entorno del escritorio para satisfacer sus preferencias.






# Para abrir el Gestor de estilos


Pulse el control del Gestor de estilos del Panel Frontal.
