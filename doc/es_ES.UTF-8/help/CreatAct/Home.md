
# Ayuda de Crear Acción
Ayuda de Crear Acción

Crear Acción es una aplicación que permite:

Crear unaacciónpara iniciar una aplicación

Crear un "data types"tipo de datos para los
archivos de datos de una aplicación

Crear acciones para abrir e imprimir los archivos de datos de la
aplicación

Crear acción también resulta útil en la creación de acciones
sencillas para los scripts de shell.
# Para ejecutar Crear Acción
Crear Acción: inicioacción: creartipo de datos: crear

Abra el grupo de aplicaciones Apls_Escritorio, que se encuentra en
el Gestor de aplicaciones, y pulse dos veces sobre Crear Acción.
