
# Ayuda del Gestor de archivos
Ayuda del Gestor de archivos

introducciónEl Gestor de archivos sirve para crear, buscar y utilizar objetos
del espacio de trabajo: archivos,
carpetas y aplicaciones. Cada objeto se representa mediante un
icono en el Gestor de archivos. El Gestor de archivos le permite
realizar una serie de acciones con estos objetos.
Por ejemplo, puede moverlos, copiarlos, abrirlos y suprimirlos.

* **Menús** 

Contiene los mandatos para el Gestor de archivos
* **Mostrar ruta con iconos** 

Utiliza iconos para mostrar la ubicación de la carpeta
que está examinando
* **Ruta de texto** 

Utiliza texto para mostrar la ubicación de la carpeta
que está examinando
* **Línea de estado** 

Muestra el nombre de ruta completo de la carpeta que
está examinando
* **Área de visualización** 

Muestra el contenido de la carpeta que está examinando
* **Iconos de objeto** 

Cada icono representa un archivo o carpeta en la
carpeta actual
* **Línea de mensajes** 

Esta línea del final de la ventana muestra cuántos
objetos hay dentro de la carpeta que está examinando.
También muestra mensajes cuando el Gestor de archivos
está ocupado realizando acciones, por ejemplo abrir
una carpeta.
