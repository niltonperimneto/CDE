&newline;To help you organize and manage your work, the desktop provides windows,
workspaces, controls, menus, and the Front Panel.

Windowscontain software applications and are
framed with controls so you can move them, size them, or place them in
additional workspaces.

Workspacesare the screen areas where you place the
windows needed for your work, arrange them, and put them away when you're done.

Controlsenable you to manipulate objects, select
choices, or type information.

Menusprovide access to commands you use to manage
windows and operate applications.

TheFront Panelis a collection of frequently used
controls, available in every workspace.
