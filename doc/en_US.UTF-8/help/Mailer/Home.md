
# Mailer Help
Mailer HelpMailer, usingusing Mailerintroduction

Use Mailer to send, receive, and manage your electronic mail messages.
When you receive an electronic message (email), you can:

Save a message to a Mailbox in your file system

Reply to the sender of the message

Forward the message to another recipient along with your own message

Delete the message

Print the message

You can send and receive attachments: image files, document files, or
executable files.

You can also drag and drop files from other desktop applications, such as
File Manager, for use as attachments in your mail messages.
# To Open Mailer


Click the Mailer control in the Front Panel.
