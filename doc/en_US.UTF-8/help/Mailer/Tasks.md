Or,

# To Sort Mail Messages
sorting messagesmessages, sorting

From the View menu, choose a sort type:

* **Use** 

To
* **Date/Time** 

Sorts messages in order of most recent at the bottom of the mailbox
* **Sender** 

Groups messages by a particular sender
* **Subject** 

Groups messages together on the same subject
* **Size** 

Groups messages from smallest to largest
* **Status** 

Groups messages together by read, unread, and new (useful
if you read your mail in non-sequential order)


The message headers are rearranged and redisplayed in the mailbox
in the new order.
# To Find a Mail Message
finding messagesmessages, finding

Choose Find from the Message menu.
&newline;The Find dialog box appears.

Type the text you want to search on in the appropriate
field.
You can search for a match on the To, From, Subject, and Cc fields.

Select Forward or Backward to set the search direction
in the mailbox.

Click Find to select the first message found.
If you think there may be several messages, click Select All
to mark (by selection) each matching message.

You can remove all the text in the Find fields at any
time by clicking Clear.

# To Compose, Address, and Send a Message
creating messagesmessages, creatingcomposing messagesmessages, composingaddressing messagesmessages, addressingsending messagesmessages, sending

Choose New Message from the Compose menu.

The (New Message) Compose window appears.

Type the email addresses of the people to whom you want to send the
message in the To field.

You can send a message to a single user, multiple users, or
analias.

Type the topic or title of the message in the Subject field.

Optional: Type the email addresses of the people whom you want to
be carbon copy recipients of the messages in the Cc field.

Type the message text in the text area.

Optional: To format the text with a 72-character right
margin, select the text and choose Word Wrap from the Compose
Edit menu.
For information on editing functions, refer to, or.

Optional: Add files to the message, if desired.

To find out how to add an attachment, see.

Click the Send button or choose Send from the Compose File menu.

The Compose window closes and the message is sent.

Clicking Close removes the Compose window and discards
your message. Be sure to save any text you might want later
before closing the Compose window.

# To Reply to a Message
replying to messagesmessages, replying toanswering messagesmessages, answering

Select the message to which you want to reply.

Determine the scope of the reply.

To reply to the sender, click Reply to Sender or choose
Reply to Sender from the Compose menu.

To reply to the sender and all the recipients of the message,
choose Reply to All from the Compose menu.

To reply to the sender and all the recipients of the message
and include a copy of the message, choose Reply to All, Include
from the Compose menu.

Type the text of the reply.

Click the Send button or choose Send from the Compose File menu.

The Compose window closes and the message is sent.

# To Undelete a Mail Message
messages, undeletingundeleting messagesmessages, recovering deletedrecovering deleted messages

If you accidentally delete a message you would
like to keep, you can undelete it.

To restore a message you just deleted, choose Undelete Last from the
Message menu. Repeatedly selecting Undelete Last will undelete
additional messages, in reverse order to the order in which they were
deleted.

To restore an earlier message, choose Undelete from List
from the Message menu, select one or more messages from the
list of deleted messages, and click Undelete.

Make the deletions in the list permanent by choosing Destroy Deleted Messages
from the Mailbox menu.

# To Drag and Drop Messages to Other Mailboxes
saving messages in other mailboxes, using drag and dropmessages, saving in other mailboxes using drag and drop

Message Header list is visible.

Open two or three frequently used mailboxes on the desktop

Select the message to be copied.

# To Create an Alias
aliases, creatingcreating aliases,electronic mail aliases, creating

You may want to create an alias for a user or group of users
to whom you frequently send mail.

Choose Alias from the Category menu of the Mail Options
dialog box.

Type the name for your alias in the Alias field.

Type the user addresses that will make up the alias.

Each of the addresses should be in the formusername@hostnameand be separated by a comma,
space, or both.

Click Add to add the information in the Alias and Addresses fields to
the Alias list.

Click OK or Apply to make the settings take effect.

You can also change or remove an alias.
