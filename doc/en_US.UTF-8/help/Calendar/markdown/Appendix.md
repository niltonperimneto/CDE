# Calendar Components
Calendar, main windowmain window

&newline;&empty;

&newline;&empty;

* **Tool bar** 

Contains the icons for opening the Appointment or
To Do Editors, and changing the Calendar view.
&newline;&empty;&newline;
* **Appt Editor icon** 

Opens the Appointment Editor.
* **To Do Editor icon** 

Opens the To Do Editor.
&newline;&empty;&newline;
* **Previous view arrow** 

Changes the Calendar view to the previous view (Month
view example: from November 1994 to October 1994).
* **Today button** 

Changes the Calendar view to show the current day.
* **Next view arrow** 

Changes the Calendar view to the next view (Week view
example: from "Week Starting Mon, October 17 1994"
to "Week Starting Mon, October 24 1994".
&newline;&empty;&newline;
* **View icons** 

Use the view icons to Change the current view
to day, week, month, or year view.
&newline;&empty;&newline;
* **View label** 

Title of day, week, month, or year displayed.
* **Calendar view** 

Main view of the Calendar.
* **Message area** 

Displays status messages from Calendar.
