Choose Find from the View menu.

In the Find field, type some text that can become a keyword for the 
search; for example,DentistorConference.

group appointment, selecting day and timeappointment, groupcompare calendars grid, usinggrid, Compare Calendarsgroup appointment, schedulingscheduling, group appointmentGroup Appointment Editor, opening

Select one or more entries in the Browse list.

calendars.

# To Mail Reminders to the Group
group appointment, mail remindersGroup Appointment Editor, mailing remindersmail reminders, of group appointmentreminders (mail), of group appointment

Once you've set up a group appointment (date, start and end times,
description, repeat interval), you may want to mail
the group a reminder.  To send mail:

Click Mail in the Group Appointment Editor.

Calendar displays a Compose window containing an appointment
attachment.  The appointment attachment is pre-addressed to the 
selected users and automatically filled out with the date and 
appointment start and end times.

Click Deliver in the Compose window.

Calendar mails the reminder to the selected users.

Recipients of a mail message with a calendar-formatted 
reminder can drag the message from the Mailer header window
and drop it on their calendars to automatically schedule the 
pre-formatted appointment.

# To Change the Default Time Zone
defaults, time zonetime zone, defaulttime zone, changingcustom time, setting

Choose Time Zone from the View menu.

The Time Zone dialog box is displayed.

Select the Custom Time setting.

In the Hours field, edit the number of hours before or 
after Greenwich Mean Time that you want your calendar set to.

For example, to synchronize your calendar for Pacific Standard Time,
choose GMT+8 hours.  For Finland, choose
GMT-2 hours.  For Japan, choose GMT-9 hours.  You may have to adjust
by one hour for daylight, or summer time.

To revert to your own time zone, select My Time.

Click OK to apply changes and close the dialog box.

# To Drag and Drop an Appointment on Your Calendar
appointment, dragging from Mailerdragging appointment from Mailer to Calendardrag and drop, Mailer to Calendar

Create an appointment using the standalone Appointment Editor.

At a command line, typedtcm_editor`filename`

The standalone Appointment Editor is displayed.

Type a date in the Date field.

Remember to use the date format set in the Date Format dialog box.

Choose the Start and End times for the appointment.

Type a description of the appointment in the What field.

Choose a frequency for the appointment from the Occurs menu.

Click Save to write the appointment contents to the file you 
specified at the command line.

Open File Manager and navigate to the folder where the appointment 
file is stored.

Drag the file icon and drop it on Calendar.

Calendar automatically schedules the appointment on your calendar.

You could also drag the file icon to Mailer and mail the formatted 
appointment to others.
