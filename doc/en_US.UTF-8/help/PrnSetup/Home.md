
# Print Setup Window
To access the Print Setup Window, click theSetup...button
on an application's print dialog box.

There are three areas on the Print Setup window:

* **Current Printer Information** 

This is the top area. It lists the name and description
for the currently selected printer.
* **Printer Options** 

This area is accessed by clicking the Printer tab. It
lists the model of the currently
selected printer and presents the options available for the printer.
* **Job Options** 

This area is accessed by clicking the Job tab. It
presents the job options that you can set.
