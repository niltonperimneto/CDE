
# Printer Information Dialog Box
The Printer Information dialog box provides a detailed description
of the currently selected printer. You can access it from either the print
dialog box or the More Printers dialog box.After viewing the printer description, click Ok to return to your
previous window.The Help button displays help on how to use the
Printer Information dialog box.
# More Printers Dialog Box


The More Printers dialog box provides a list of the available printers.To select a printer from the list, click on its name and then
click the Ok button. The More Printers dialog closes and you
are returned to the print dialog box, with the selected printer
replacing the previous selection (if any).The More Printers dialog box contains the following buttons:

* **Ok** 

Establishes the selected printer as the current selection and returns you
to the print dialog box.
* **Info...** 

Displays the Printer Information dialog box, showing a detailed
description of the currently highlighted printer. See. After viewing the description, click
Ok to return to the More Printers dialog box.
* **Cancel** 

Cancels the printer selection operation and returns you to the
print dialog box.
* **Help** 

Displays help information about the
More Printers dialog box.

# File Selection Dialog Box


The File Selection dialog box allows you to select a
print output file by browsing through the directory
structure. You can select either a new or existing file.After specifying the file, click the Ok button.
The File Selection dialog closes and you
are returned to the print dialog box, with the selected file
replacing the previous selection.

* **Enter path or folder name** 

Identifies the path name of the current folder.
* **Filter** 

An asterisk (*) shows all files. You can enter
wildcard characters to display only those files that
match an extension. For example, *.ps lists
only those files with a .ps extension.
* **Files** 

Lists files located in the current folder.
* **Folders** 

Lists folders located in the current folder. You can select a file in the
current folder, a subfolder, or a different folder.
* **Enter a file name** 

Displays the file name selected in the Files list.
Press Return or click Ok to select the file. Or,
you can type the name of the file you want to select.
* **OK** 

Establishes the selected file as the current selection and returns you
to the print dialog box.
* **Update** 

Displays a new list of files after
changing the filter key or changing to a new folder.
* **Cancel** 

Cancels the file selection operation.
* **Help** 

Displays help information about the
File Selection dialog box.
