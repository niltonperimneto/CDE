
# Glossary
architectural formA template for a class of SGML elements or set of
attributes.attributeThe named value associated with a specific occurrence of
an element.bookIn the Information Manager, an electronic book. A collection of sections
arranged according to a hypertext table of contents (TOC).bookcaseA group of electronic books on related topics. A logical set
of information (perhaps several related books) that are
accompanied by a single full-text index. A bookcase and
style sheets represents the minimum collection of
information that can be built into an
Information Library information
library.Bookcase DTDThe document type definition defining the structure of
the specification files used to build books for the
Information Manager.bookcase specificationAn SGML document that contains or specifies the book
files, style sheets, and other SGML entities necessary to
build an Information Manager information library.Book ListThe list of bookcases and books available from the
Information Manager.BrowserThe Information Manager on-line system that allows
you to view and manage on-line information.document instanceThe SGML source for your document, including the
document content and the SGML markup.DTDAcroynm for document type definition. The formal
constructs that establish the markup for a document.

architectural forms applied.reading windowThe Information Manager window that shows the text, tables
