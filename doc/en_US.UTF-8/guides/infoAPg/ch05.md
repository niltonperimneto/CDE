
# Administering Bookcasesinformation librariesbookcase administration

After building bookcases into an information library, you can
perform several administrative functions on the bookcases that
make up the library without having to rebuild them. Specifically,
you can:

Copy bookcases from another library

Remove bookcases from a library

List the bookcases in a library

Rename bookcases in a library

Rearrange bookcases in a library

See alsoAdding a Bookcase to an Existing Library.
# Running dtdocbook2infolib admin
Information Manager Toolkitcommandsdtdocbook2infolib admin

dtdocbook2infolib adminis an interactive program that displays a menu
from which you select the administrative operation you want to
perform. To run thedtdocbook2infolib admincommand,
on the command line type:dtdocbook2infolib admin

When you press enter, the following menu appears.1) Copy a bookcase from another library
2) Remove a bookcase
3) List bookcases in a library
4) Rename a bookcase
5) Rearrange bookcases in a library
6) Exit
Please enter your choice [1-6]
# Copying Bookcases from Another Library
information librariescopying bookcases betweenbookcasescopying from another library

When you copy a bookcase from another library,dtdocbook2infolib adminvalidates the bookcase you are copying against the bookcases in
the destination library. This ensures that no section IDs are
duplicated within the library, and that the database versions are
compatible with each other.

If the bookcase you want to copy already exists in the destination
library,dtdocbook2infolib adminasks whether you wish to replace the
bookcase.

To copy a bookcase from another library:

At a command line, enterdtdocbook2infolib admin.

The following menu appears:1) Copy a bookcase from another library
2) Remove a bookcase
3) List bookcases in a library
4) Rename a bookcase
5) Rearrange bookcases in a library
6) Exit
Please enter your choice [1-6]

On the command line, enter1.

dtdocbook2infolib admindisplays:Enter the path for the library from which you want to copy a bookcase.
This can be a relative or absolute path.

Enter the path to the information library from which you
want to copy a bookcase. For example:/lit/wolfe

dtdocbook2infolib admindisplays the bookcases available in the
library you specified. For example:The bookcases available in /lit/wolfe are:

 1) Life [Life Works]
 2) Essays [Essays from Life]
 3) Poems [Collections of Poems]

Enter the number associated with the bookcase you want to copy.

At the prompt, enter the number corresponding to the
bookcase you want to copy.dtdocbook2infolib admindisplays:You specified[Life]. Do you really want to copy this bookcase?

To copy the bookcase you specified, enterY(you can use
an uppercase or lowercase character).

Enter the path to the information library to which you
want to copy the bookcase. For example:/lit/destiny

dtdocbook2infolib admindisplays the bookcases available in the
library you specified:The bookcases available in /lit/destiny are:

 1) Child [School Days]
 2) Reports [Reports from School]

Bookcases appear in the browser Book List in the order listed here.
Enter the number corresponding to the position you want bookcase to occupy [1-2].

Enter the number corresponding to the position you want
the bookcase to occupy.

If another bookcase already occupies the position you
choose, the bookcase you are copying takes its place, and
the original bookcase moves down one place in the list.

dtdocbook2infolib admindisplays the new order of the bookcases in
the library. For example:You specified this order for the bookcases:

 1) Child [School Days]
 2) Reports [Reports from School]
 3) Life [Life Works]

Is this correct? [ynq]

To copy the bookcase into this position, enterY.dtdocbook2infolib adminsaves the addition to the library and returns
to the main menu.
# Removing Bookcases from a Library
information librariesremoving bookcases frombookcasesremoving from a library

The procedure for removing a bookcase withdtdocbook2infolib
adminis similar to the procedure for copying a bookcase into
a library. When you remove a bookcase from a library, its database is
deleted from the information library.

When you specify the library to work with,dtdocbook2infolib
adminlists the bookcases found in that library. You simply
specify the bookcase to remove. If the information you supply is valid,dtdocbook2infolib adminprompts you for confirmation before
removing the bookcase you specified.

To remove a bookcase from a library:

At a command line, enterdtdocbook2infolib admin.

The following menu appears.1) Copy a bookcase from another library
2) Remove a bookcase
3) List bookcases in a library
4) Rename a bookcase
5) Rearrange bookcases in a library
6) Exit
Please enter your choice [1-6]

To remove a bookcase, enter2.

dtdocbook2infolib admindisplays:Enter the path for the library from which you want to remove a bookcase.
This can be a relative or absolute path.

Enter the path to the information library from which you
want to remove the bookcase. For example:/lit/wolfe.

dtdocbook2infolib admindisplays the bookcases available in the
library you specified:The bookcases available in /lit/wolfe are:

1) Life [Life Works]
2) Essays [Essays from Life]
3) Poems [Collections of Poems]

Enter the number corresponding to the bookcase to remove.

At the prompt, enter the number corresponding to the
bookcase you want to remove.dtdocbook2infolib admindisplays:You specified[Essays]. Do you really want to remove this bookcase?

To remove the bookcase you specified, enterY.dtdocbook2infolib adminremoves the bookcase and returns to the
main menu.

# Rearranging Bookcases in a Library
information librariesrearranging bookcases inbookcasesrearranging in a library

After you have built a library, you may find that you want to
display the bookcases in a different order in the Information Manager Book List.
Or, you may have added a bookcase usingdtdocbook2infolib build, and
want to move it to a different position. You can usedtdocbook2infolib adminto
change the order of bookcases in a library.

To rearrange bookcases within a library:

At a command line, enterdtdocbook2infolib admin.

dtdocbook2infolib admindisplays:1) Copy a bookcase from another library
2) Remove a bookcase
3) List bookcases in a library
4) Rename a bookcase
5) Rearrange bookcases in a library
6) Exit
Please enter your choice [1-6]

To rearrange bookcases within a library, enter5.

dtdocbook2infolib admindisplays:Enter the path for the library you want to rearrange.
This can be a relative or absolute path.

Enter the path to the information library you want to
rearrange. For example:/lit/wolfe.

dtdocbook2infolib admindisplays the bookcases available in the
library you specified:The bookcases available in /lit/wolfe are:

1) Life [Life Works]
2) Essays [Essays from Life]
