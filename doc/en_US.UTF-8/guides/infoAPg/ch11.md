
# Table of Contents Architectural Forms
architectural formstable of contents

If you already have a table of contents document instance in
SGML format, you do not have to create a new table of contents.
Instead, you can apply
Information Manager architectural forms to the
elements in your existing table of contents DTD so
Information Manager knows
how to treat the table of contents information.

For information on creating an SGML table of
contents that conforms to the
Information ManagerdtinfoTOC.dtd,
seeCreating a Table of Contents.

You can add these attributes to your existing table of contents
DTD:

* **`DTINFO.TOC`** 

Apply`DTINFO.TOC`to elements that designate tables of
contents.

Entity name:`DTINFO.TOC`

Attribute template:`DTINFO.TOC CDATA #FIXED`

Related Forms:`DTINFO.Scope`,`DTINFO.Value`,`DTINFO.ID`.

Example:&lt;!ELEMENT TOC - - ( Title, TitleAbbrev?, TOCfront*,
                        (TOCpart+ | TOCchap+), TOCback* ) &gt;
&lt;!ATTLIST TOC
        Id              ID      #REQUIRED

        %DTINFO.TOC;            &ldquo;&num;CONTENT&rdquo;
        %DTINFO.ID;             &ldquo;attr( Id )&rdquo;--&gt;
&gt;`DTINFO.TOCEntry`

Apply`DTINFO.TOCEntry`to elements that indicate an entry in
a book&rsquo;s table of contents.

Entity name:`DTINFO.TOCEntry`

Attribute template:`DTINFO.TOCEntry CDATA #FIXED`

Related Forms:`DTINFO.Scope`,`DTINFO.Value`,`DTINFO.ID`.

Example:&lt;!ELEMENT TOCpart - - ( TOCentry+, TOCchap* ) &gt;
&lt;!ATTLIST TOCpart
        Id              ID      #IMPLIED

        %DTINFO.ID;             &ldquo;attr( Id )&rdquo;
        %DTINFO.TOCEntry;       #CONTENT&rdquo;
&gt;`DTINFO.TOCLevel`Apply`DTINFO.TOCLevel`to elements that indicate an entry
in a book&rsquo;s table of contents.Entity name:`DTINFO.TOCLevel`Attribute template:`DTINFO.TOCLevel CDATA &num;FIXED`Related Forms:`DTINFO.Scope`,`DTINFO.Value`,`DTINFO.ID`.Example:&lt;!ELEMENT TOCpart - - ( TOCentry+, TOCchap* ) &gt;
&lt;!ATTLIST TOCpart
        Id              ID      #IMPLIED

        %DTINFO.ID;             &ldquo;attr( Id )&rdquo;
        %DTINFO.TOCLevel;
&gt;
