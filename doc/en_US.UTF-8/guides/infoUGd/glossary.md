
# Glossary
Auto TrackBook List window: Options menu: Use to control whether or not the
Information Manager shows you where you are in the information library
hierarchy.bookAn electronic book. See alsobookcaseandinformation library.bookcaseA group of electronic books on related topics. A logical set of
information (perhaps several related books) that is accompanied by a
single full-text index. A bookcase and style sheets represent the
minimum collection of information that can be built into an information
library. See alsoinformation library.bookcase DTDThe document type definition that defines the structure of the
specification files used to build books for the Information Manager. See
alsodocument type definition.bookcase specificationAn SGML document that contains or specifies the book files, style
sheets, and other SGML entities necessary to build an information
library.book listThe list of bookcases and books available from the Information Manager.Book List windowThe Information Manager dialog box that provides access to information
stored in an information library for on-line display or printing.bookmarkAn Information Manager marker that users can create to annotate specific
text or graphics in an information library. The Information Manager
represents a bookmark as an icon in the left margin of a reading window.Bookmark EditorThe dialog box that appears when you double-click on a bookmark icon in
a Reading window. Use to view and edit existing bookmarks and notes.browse1) To scan a collection, such as a database, a File Manager view, a list
box, or a text file, either for a particular item or for anything that
seems to be of interest; implies observing rather than changing
information.2) Searching by examining and scrolling through screens or windows of
data.connector buttonQuery Editor: Use to include another term in a search.Create BookmarkReading window: Marks menu: Use to create a blank bookmark within a
section in an active reading window.Create NoteReading window: Marks menu: Use to create an annotated bookmark within a
section in an active Reading window.Detach Graphicmenu item: Use to copy a selected graphic image into a separate window
called a Detached Graphic window.Detached Graphic iconAn icon that marks where a graphic has been detached.Detached Graphic windowThe window dialog box that appears when you detach a graphic.Front Panel controlFront Panel: An object in the Front Panel used as an interface to basic
system services and frequently performed tasks and operations. Use the
specific control name where appropriate. Controls in the default Front
Panel are: the Clock control, the Calendar control, the File Manager
control, the Text Editor control, the Mailer control, the Printer
control, the Style Manager control, the Application Manager control, the
Information Manager control, and the Trash Can control.graphical mapA visual representation of a book's organizational structure in which
the book's section titles are displayed as a graphical "tree".GoReading window: Go menu: Use to navigate sequentially through an
information library or to return to sections you've previously read.hypertext table of contentsA hypertext hierarchical list of section titles. Hypertext TOCs organize
sections of online book titles within the Information System.infolibThe set of files, including SGML instances, graphics, and indexes, that
comprise an information library. Seeinformation
library.Info ManagerSeeInformation Manager.information libraryA set of bookcases, where each bookcase contains a set of books, and
where each book contains sections.Information ManagerThe software application that permits you to browse and search collections
of online documentation, called information libraries.Information SystemThe information libraries (the data), the Information Manager (the viewer)
and the Information System Developer's Toolkit (the development tools).Information System Developer's ToolkitThe set of tools and APIs that allow Information System authors
to write, process, and view information libraries.linkA hypertext reference to a locator. The content of a link represents an
active area within the Information Manager. Seelocator.List MarksReading window: Marks menu: Use to list all of your blank and annotated
bookmarks.locatorAn anchor point in a document to which hypertext links can refer. Each
locator has an associated identifier that is unique throughout the
information library.MarksReading window: Marks menu: Use to create, edit, and delete bookmarks.panner boxA two-dimensional interactive panner within an Information Manager
Graphical Map that allows you to shift the part of the map that is
visible within the reading window.pinned windowA reading window that is marked so that the text displayed in the window
is retained when a link is followed, and a new window is opened for the
new text. If a window is not pinned, the Information Manager updates the
text in the current reading window each time a link is followed.Print SetupA dialog box that allows you to specify printer settings for the
currently selected printer, such as page orientation (portrait or
landscape), one- or two-sided printing, or which paper tray to use.
Calendar, Information Manager, Mailer, and Text Editor display this
dialog box when you double-click on the Setup button in their print
dialog box.Query EditorThe dialog box that allows you to define detailed searches of an
information library.Reading windowThe Information Manager window that shows the text, tables, and graphics
of a particular section in a book.Search Results windowA window that displays a list of sections that contain search matches.
The Information Manager displays a Search Results window upon completion
of a search.search scopeThe scope of books and types of information that an Information Manager
query will search.Search Scope EditorThe dialog box that allows you to create custom search scopes.sectionA single unit of documentation that is displayed within an Information
Manager Reading window.ViewAn icon and a menu item that allows you to open a Reading window on a
selected section.
