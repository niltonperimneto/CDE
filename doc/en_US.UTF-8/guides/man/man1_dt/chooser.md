# dtchooser
user cmddtchooserCDE dtlogin chooser screen display utilitydtchooser
## DESCRIPTION


The CDE dtchooser utility is used bydtloginto display the host chooser screen on a display when an`XDMCP``indirect`request is received. The user can select a host from the
list presented, anddtloginon that host will display a login screen on the display. There will be onedtchooserprocess per display on which a chooser screen is visible.

All customization of the chooser screen is done viadtloginconfiguration files. Refer to the
&cdeman.dtlogin; man page for information regarding chooser screen customization.
