## DESCRIPTION


`XmCreateDrawingArea`creates an instance of a DrawingArea widget and returns the
associated widget ID.

* **`parent`** 

Specifies the parent widget ID
* **`name`** 

Specifies the name of the created widget
* **`arglist`** 

Specifies the argument list
* **`argcount`** 

Specifies the number of attribute/value pairs in the argument list
(`arglist`)


For a complete definition of DrawingArea and its associated resources, see
&cdeman.XmDrawingArea;.
## RETURN


Returns the DrawingArea widget ID.
