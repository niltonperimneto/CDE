## DESCRIPTION


XmTabis a data structure that specifies a tab stop to be used
in rendering anXmStringcontaining tab components. AnXmTabvalue contains a value, a unit type, an offset model
(either`XmABSOLUTE`or`XmRELATIVE`), an alignment model
(`XmALIGNMENT_BEGINNING`),
and a decimal point
character. The resource file syntax forXmTabis specified in
theXmTabListreference page.
