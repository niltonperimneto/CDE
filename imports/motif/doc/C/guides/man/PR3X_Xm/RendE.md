## DESCRIPTION


`XmRenditionUpdate`modifies resources in the specified rendition.

* **`rendition`** 

Specifies the rendition.
* **`arglist`** 

Specifies the argument list.
* **`argcount`** 

Specifies the number of attribute/value pairs in the argument list
(`arglist`).
