# XmSimpleSpinBoxAddItem
library call`XmSimpleSpinBoxAddItem`add an item to the XmSimpleSpinBox#include <Xm/SSpinB.h>void`XmSimpleSpinBoxAddItem`WidgetwXmStringitemintpos
## DESCRIPTION


The`XmSimpleSpinBoxAddItem`function adds the given item to the XmSimpleSpinBox at the given
position.

The`w`argument specifies the widget ID.

The`item`argument specifies theXmStringfor the new item.

Theposargument specifies the position of the new item.
## RETURN VALUE


The`XmSimpleSpinBoxAddItem`function returns no value.
